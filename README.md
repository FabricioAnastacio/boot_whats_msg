## Robo de menssagens

Este robo é capas de acessar sua conta no Whats* e mandar varias mensagens com um texto e um curriculo ou qualquer outro arquivo para varias pessoas com seu numero listado em uma lista
em excel

## Como usar

- Baixe o repositorio, crie o ambiente virtual, instale as dependencias e rode o codigo

- Baixe e adicione ao .venv/scripts o executavel chromedriver.exe; Ele que fara o acesso ao browser

- Adicione no terminal o caminho ate o arquivo que sera enviado no formato: D:\exemple\exemple\arquivo .pdf .odf .txt .word .excel

- Escaneie o qr code que aparece

## Explicação

É usado o Chrome para a pesquisa; O Chrome deve estar na mesma versão do chromedriver que possivelmente sera a versão 114.*

É esperado que contenha pelomenos um numero na lista Enviar.xlsx
